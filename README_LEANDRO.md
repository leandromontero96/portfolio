## 🚀 Cómo Usar Este Portfolio

### Visualización Local

1. **Abrir directamente en navegador**:
   ```bash
   # Navega a la carpeta
   cd portfolio-it

   # Abre index.html con tu navegador favorito
   # Doble click en index.html
   ```

2. **Usar servidor local (opcional)**:
   ```bash
   # Con Python
   python -m http.server 8000
   # Abre http://localhost:8000

   # O con Live Server en VS Code
   # Click derecho en index.html > Open with Live Server
   ```

### Personalización Adicional

#### Cambiar Colores
Edita `css/styles.css` líneas 3-20:
```css
:root {
    --primary-color: #00d4ff;      /* Color principal */
    --secondary-color: #7b2cbf;    /* Color secundario */
    --accent-color: #ff006e;       /* Color de acento */
}
```

#### Agregar Tu Foto
1. Guarda tu foto en `assets/images/perfil.jpg`
2. Edita `index.html` línea 82-86:
```html
<div class="about-image">
    <img src="assets/images/perfil.jpg" alt="Leandro Montero">
</div>
```

#### Actualizar Proyectos con Enlaces Reales
Cuando tengas repositorios de GitHub listos, actualiza los enlaces en `index.html` sección de proyectos (líneas 298-427).

## 🌐 Deployment

### GitHub Pages (Recomendado)

1. **Crear repositorio en GitHub**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Portfolio Leandro Montero"
   git branch -M main
   git remote add origin https://github.com/leandromontero96/portfolio.git
   git push -u origin main
   ```

2. **Activar GitHub Pages**:
   - Ve a Settings > Pages
   - Source: Deploy from a branch
   - Branch: main / root
   - Save

3. **Tu portfolio estará en**:
   `https://leandromontero96.github.io/portfolio/`

### Netlify (Alternativa)

1. **Drag & Drop**:
   - Ve a [netlify.com](https://netlify.com)
   - Arrastra la carpeta `portfolio-it`
   - ¡Listo!

2. **O con CLI**:
   ```bash
   npm install -g netlify-cli
   netlify deploy --prod
   ```

### Vercel (Alternativa)

```bash
npm install -g vercel
cd portfolio-it
vercel
```
